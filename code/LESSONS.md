# 错误教训库（Lessons Learned）

> **本文件是 Claude Code 的强制记忆文件。**
> 每次生成代码前必须检查本文件，避免重复犯错。
> 用户报告的 Bug 修复后，由 Claude 自动追加到对应分类。

---

## 使用说明

**触发条件：**
- 用户说"这个有问题"、"上次犯过"、"不要再犯"、"记住这个错误"
- 修复 Bug 后，Claude 主动记录

**记录格式：**
每条记录必须包含：日期、错误描述、错误代码、正确代码、根因、教训

## 错误记录模板

添加新记录时，复制以下模板粘贴到对应分类下：

```
### [日期] 简短标题

**问题描述：**
（一句话描述 Bug 现象）

**错误代码：**
```typescript
// 出错的代码片段
```

**正确代码：**
```typescript
// 修复后的代码片段
```

**根因：**
（为什么会出错，底层原因是什么）

**教训：**
（一句话总结，格式：**永远不要 XXX，应该 XXX**）

**影响范围：**
（涉及哪些文件/模块，修复是否可能影响其他地方）
```
